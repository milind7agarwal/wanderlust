#  Wanderlust – Hotel Listing Web Application

Wanderlust is a full-stack hotel listing platform where users can explore, create, edit, and manage property listings. The project demonstrates strong backend fundamentals, RESTful routing, authentication, and database management using the MERN-related stack (without React).

---

##  Features

-  Create, Read, Update, Delete (CRUD) hotel listings  
-  User Authentication & Authorization  
-  Add and manage listings securely  
-  Flash messages for better user experience  
-  RESTful routing architecture  
-  Image upload support  
-  Server-side validation & middleware protection  
-  Structured MVC project architecture  

---

##  Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Frontend
- EJS (Embedded JavaScript Templates)
- Bootstrap / CSS

### Authentication & Security
- Passport.js
- Express-session
- Connect-flash
- Joi (for validation)
